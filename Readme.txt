Trabalho Desenvolvido na Disciplina de Desenvolvimento de Sistemas Distribuidos.

Ministrada pelo Prof. Robson Fernandes (Unip Bauru/2020).

O Trabalho consiste em um classificador de letras de músicas, onde através do Scrapping buscamos algumas bandas pré definidas e a PipeLine faz a análise nos trazendo o resultado de qual banda seria a letra dentro de um dataset já definido também.

Em cada etapa foi procurado deixar explicado o que é realizado naquele blobo de código.

Dentro da pasta client temos um pequeno HTML para realizar a procurar via navegador, desde que a letra seja tratada e esteja sem espaçamento e sem quebras de linhas.

API publicada na Heroku pelo endereço: https://trabalhodsdia.herokuapp.com/.
